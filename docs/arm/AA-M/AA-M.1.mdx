<p>In addition to implementation-defined characteristics, each Ada implementation is required to  document various properties of the implementation: <br /></p>

<AnnotatedOnly>
<MarginText>1.a/2</MarginText>
<Admonition type="aarm" aarm="ramification">
<strong></strong>Most of the items in this list require documentation only for implementations that conform to Specialized Needs Annexes. <br /></Admonition>
</AnnotatedOnly>

<ul>
<li>The behavior of implementations in implementation-defined situations shall be documented &mdash; see <a href="../AA-M/AA-M.2">M.2</a> for a listing. See <a href="../AA-1/AA-1.1#Subclause_1.1.3">1.1.3</a>(19).<br /></li>
<MarginText>3/2</MarginText>
