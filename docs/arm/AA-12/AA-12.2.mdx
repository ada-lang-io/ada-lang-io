<MarginText>1</MarginText>
<p>The body of a generic unit (a <em>generic body</em>) [is a template for the instance bodies. The syntax of a generic body is identical to that of a nongeneric body]. <br /></p>

<AnnotatedOnly>
<MarginText>1.a</MarginText>
<Admonition type="aarm" aarm="ramification">
<strong></strong>We also use terms like &ldquo;generic function body&rdquo; and &ldquo;nongeneric package body&rdquo;. <br />
</Admonition>
</AnnotatedOnly>


#### Dynamic Semantics

<MarginText>2</MarginText>
<p>The elaboration of a generic body has no other effect than to establish that the generic unit can from then on be instantiated without failing the Elaboration{"_"}Check. If the generic body is a child of a generic package, then its elaboration establishes that each corresponding declaration nested in an instance of the parent (see <a href="../AA-10/AA-10.1#Subclause_10.1.1">10.1.1</a>) can from then on be instantiated without failing the Elaboration{"_"}Check. <br /></p>

<AnnotatedOnly>
<MarginText>3</MarginText>
<Admonition type="aarm" aarm="note">
NOTE   The syntax of generic subprograms implies that a generic subprogram body is always the completion of a declaration. <br />
</Admonition>
</AnnotatedOnly>


#### Examples

<MarginText>4</MarginText>
<p><em>Example of a generic procedure body:</em> <br /></p>

<MarginText>5</MarginText>
<CodeBlock language="ada">
procedure Exchange(U, V : in out Elem) is  -- see <a href="../AA-12/AA-12.1">12.1</a>{"\n"}   T : Elem;  --  the generic formal type{"\n"}begin{"\n"}   T := U;{"\n"}   U := V;{"\n"}   V := T;{"\n"}end Exchange;{"\n"}

</CodeBlock>
<MarginText>6</MarginText>
<p><em>Example of a generic function body:</em> <br /></p>

<MarginText>7</MarginText>
<CodeBlock language="ada">
function Squaring(X : Item) return Item is  --  see <a href="../AA-12/AA-12.1">12.1</a>{"\n"}begin{"\n"}   return X{"*"}X;  --  the formal operator "{"*"}"{"\n"}end Squaring;{"\n"}

</CodeBlock>
<MarginText>8</MarginText>
<p><em>Example of a generic package body:</em> <br /></p>

<MarginText>9</MarginText>
<CodeBlock language="ada">
package body On{"_"}Vectors is  --  see <a href="../AA-12/AA-12.1">12.1</a>{"\n"}
<MarginText>10</MarginText>
   function Sum(A, B : Vector) return Vector is{"\n"}      Result : Vector(A'Range); --  the formal type Vector{"\n"}      Bias   : constant Integer := B'First - A'First;{"\n"}   begin{"\n"}      if A'Length /= B'Length then{"\n"}         raise Length{"_"}Error;{"\n"}      end if;{"\n"}
<MarginText>11</MarginText>
      for N in A'Range loop{"\n"}         Result(N) := Sum(A(N), B(N + Bias)); -- the formal function Sum{"\n"}      end loop;{"\n"}      return Result;{"\n"}   end Sum;{"\n"}
<MarginText>12</MarginText>
   function Sigma(A : Vector) return Item is{"\n"}      Total : Item := A(A'First); --  the formal type Item{"\n"}   begin{"\n"}      for N in A'First + 1 .. A'Last loop{"\n"}         Total := Sum(Total, A(N)); --  the formal function Sum{"\n"}      end loop;{"\n"}      return Total;{"\n"}   end Sigma;{"\n"}end On{"_"}Vectors;{"\n"}

</CodeBlock>
