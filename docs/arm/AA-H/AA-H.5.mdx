<MarginText>1/5</MarginText>
<MarginInfo items={["AI95-00305-01", "AI12-0267-1"]} />
<p>The following <code><a href="../AA-2/AA-2.8#S0019">pragma</a></code> requires  an implementation to detect potentially blocking operations during the execution of  a protected operation or a parallel construct. <br /></p>


#### Syntax

<MarginText>2/2</MarginText>
<MarginInfo items={["AI95-00305-01"]} />
<p>The form of a <code><a href="../AA-2/AA-2.8#S0019">pragma</a></code> Detect_Blocking is as follows: <br /></p>

<MarginText>3/2</MarginText>
<p>  <strong>pragma</strong> Detect_Blocking; <br /></p>


#### Post-Compilation Rules

<MarginText>4/2</MarginText>
<MarginInfo items={["AI95-00305-01"]} />
<p>A <code><a href="../AA-2/AA-2.8#S0019">pragma</a></code> Detect_Blocking is a configuration pragma. <br /></p>


#### Dynamic Semantics

<MarginText>5/5</MarginText>
<MarginInfo items={["AI95-00305-01", "AI12-0247-1", "AI12-0267-1"]} />
<p>An implementation is required to detect a potentially blocking operation that occurs during the execution of  a protected operation or a parallel construct defined within a compilation unit to which the pragma applies, and to raise Program_Error (see <a href="../AA-9/AA-9.5">9.5</a> ). <br /></p>


#### Implementation Permissions

<MarginText>6/5</MarginText>
<MarginInfo items={["AI95-00305-01", "AI12-0267-1"]} />
<p>An implementation is allowed to reject a <code><a href="../AA-10/AA-10.1#S0286">compilation_unit</a></code> to which a pragma Detect_Blocking applies if a potentially blocking operation is present directly within an <code><a href="../AA-9/AA-9.5#S0260">entry_body</a></code>,  the body of a protected subprogram, or a parallel construct occurring within the compilation unit. <br /></p>

<AnnotatedOnly>
<MarginText>7/5</MarginText>
<MarginInfo items={["AI95-00305-01", "AI12-0442-1"]} />
<Admonition type="aarm" aarm="note">
NOTE 1   An operation that causes a task to be blocked within a foreign language domain is not defined to be potentially blocking, and is unlikely to  be detected. <br /></Admonition>
</AnnotatedOnly>
