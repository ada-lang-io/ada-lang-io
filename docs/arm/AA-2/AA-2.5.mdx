<MarginText>1</MarginText>
<p>[A <code><a href="../AA-2/AA-2.5#S0015">character_literal</a></code> is formed by enclosing a graphic character between two apostrophe characters.] <br /></p>


#### Syntax

<MarginText>2</MarginText>

<CodeBlock>
<code>character_literal</code><a id="S0015"></a><code> ::= </code>'<code>graphic_character</code>'<br />

</CodeBlock>
<AnnotatedOnly>
<MarginText>3</MarginText>
<Admonition type="aarm" aarm="note">
NOTE   A <code><a href="../AA-2/AA-2.5#S0015">character_literal</a></code> is an enumeration literal of a character type. See <a href="../AA-3/AA-3.5#Subclause_3.5.2">3.5.2</a>. <br /></Admonition>
</AnnotatedOnly>
