#### Static Semantics

<MarginText>1</MarginText>
<p>The text of a program consists of the texts of one or more <code><a href="/docs/arm/AA-10/AA-10.1#S0285">compilation</a></code>s. The text of each <code><a href="/docs/arm/AA-10/AA-10.1#S0285">compilation</a></code> is a sequence of separate <i>lexical elements</i>. Each lexical element is formed from a sequence of characters, and is either a delimiter, an <code><a href="/docs/arm/AA-2/AA-2.3#S0002">identifier</a></code>, a reserved word, a <code><a href="/docs/arm/AA-2/AA-2.4#S0006">numeric_literal</a></code>, a <code><a href="/docs/arm/AA-2/AA-2.5#S0015">character_literal</a></code>, a <code><a href="/docs/arm/AA-2/AA-2.6#S0016">string_literal</a></code>, or a comment. The meaning of a program depends only on the particular sequences of lexical elements that form its <code><a href="/docs/arm/AA-10/AA-10.1#S0285">compilation</a></code>s, excluding <code><a href="/docs/arm/AA-2/AA-2.7#S0018">comment</a></code>s.<br /></p>

<MarginText>2/3</MarginText>
<MarginInfo items={["AI95-00285-01", "AI05-0262-1"]} />
<p>The text of a <code><a href="/docs/arm/AA-10/AA-10.1#S0285">compilation</a></code> is divided into <i>lines</i>. In general, the representation for an end of line is implementation defined. However, a sequence of one or more <code>format{"_"}effector</code>s other than the character whose code point is 16{"#"}09{"#"} (CHARACTER TABULATION) signifies at least one end of line. <br /></p>

<AnnotatedOnly>
<MarginText>2.a</MarginText>
<Admonition type="aarm" aarm="implementation-defined">
<b></b>The representation for an end of line.<br />
</Admonition>
</AnnotatedOnly>

<MarginText>3/2</MarginText>
<MarginInfo items={["AI95-00285-01"]} />
<p>[In some cases an explicit <i>separator</i> is required to separate adjacent lexical elements.] A separator is any of a <code>separator{"_"}space</code>, a <code>format{"_"}effector</code>, or the end of a line, as follows: <br /></p>

<MarginText>4/2</MarginText>
<MarginInfo items={["AI95-00285-01"]} />
<ul>
<li>A <code>separator{"_"}space</code> is a separator except within a <code><a href="/docs/arm/AA-2/AA-2.7#S0018">comment</a></code>, a <code><a href="/docs/arm/AA-2/AA-2.6#S0016">string_literal</a></code>, or a <code><a href="/docs/arm/AA-2/AA-2.5#S0015">character_literal</a></code>.<br /></li>
<MarginText>5/3</MarginText>
<MarginInfo items={["AI95-00285-01", "AI05-0262-1"]} />
<li>The character whose code point is 16{"#"}09{"#"} (CHARACTER TABULATION) is a separator except within a <code><a href="/docs/arm/AA-2/AA-2.7#S0018">comment</a></code>.<br /></li>
<MarginText>6</MarginText>
<li>The end of a line is always a separator. <br /></li>
</ul>
<MarginText>7</MarginText>
<p>One or more separators are allowed between any two adjacent lexical elements, before the first of each <code><a href="/docs/arm/AA-10/AA-10.1#S0285">compilation</a></code>, or after the last. At least one separator is required between an <code><a href="/docs/arm/AA-2/AA-2.3#S0002">identifier</a></code>, a reserved word, or a <code><a href="/docs/arm/AA-2/AA-2.4#S0006">numeric_literal</a></code> and an adjacent <code><a href="/docs/arm/AA-2/AA-2.3#S0002">identifier</a></code>, reserved word, or <code><a href="/docs/arm/AA-2/AA-2.4#S0006">numeric_literal</a></code>.<br /></p>

<MarginText>7.1/3</MarginText>
<MarginInfo items={["AI05-0079-1"]} />
<p>One or more <code>other{"_"}format</code> characters are allowed anywhere that a separator is[; any such characters have no effect on the meaning of an Ada program].<br /></p>

<MarginText>8/2</MarginText>
<MarginInfo items={["AI95-00285-01"]} />
<p>A <i>delimiter</i> is either one of the following characters: <br /></p>

<MarginText>9/5</MarginText>
<MarginInfo items={["AI12-0125-3", "AI12-0212-1"]} />
<p class="Indented2">&    '    (    )    {"*"}    +    ,    &ndash;    .    /    :    ;    {"<"}    =    {">"}    @    [    ]    |<br /></p>

<MarginText>10</MarginText>
<p>or one of the following <i>compound delimiters</i> each composed of two adjacent special characters <br /></p>

<MarginText>11</MarginText>
<p class="Indented2">={">"}    ..    {"*"}{"*"}    :=    /=    {">"}=    {"<"}=    {"<"}{"<"}    {">"}{">"}    {"<"}{">"}<br /></p>

<MarginText>12</MarginText>
<p>Each of the special characters listed for single character delimiters is a single delimiter except if this character is used as a character of a compound delimiter, or as a character of a <code><a href="/docs/arm/AA-2/AA-2.7#S0018">comment</a></code>, <code><a href="/docs/arm/AA-2/AA-2.6#S0016">string_literal</a></code>, <code><a href="/docs/arm/AA-2/AA-2.5#S0015">character_literal</a></code>, or <code><a href="/docs/arm/AA-2/AA-2.4#S0006">numeric_literal</a></code>.<br /></p>
