<MarginText>1</MarginText>
<p>[An <code><a href="../AA-5/AA-5.3#S0175">if_statement</a></code> selects for execution at most one of the enclosed <code>sequences_of_statements</code>, depending on the (truth) value of one or more corresponding <code><a href="../AA-4/AA-4.5#S0150">condition</a></code>s.] <br /></p>


#### Syntax

<MarginText>2</MarginText>

<CodeBlock>
<code>if_statement</code><a id="S0175"></a><code> ::= </code><br />    <strong>if</strong> <code><a href="../AA-4/AA-4.5#S0150">condition</a></code> <strong>then</strong><br />      <code><a href="../AA-5/AA-5.1#S0166">sequence_of_statements</a></code><br />   {"{"}<strong>elsif</strong> <code><a href="../AA-4/AA-4.5#S0150">condition</a></code> <strong>then</strong><br />      <code><a href="../AA-5/AA-5.1#S0166">sequence_of_statements</a></code>{"}"}<br />   [<strong>else</strong><br />      <code><a href="../AA-5/AA-5.1#S0166">sequence_of_statements</a></code>]<br />    <strong>end</strong> <strong>if</strong>;<br />

</CodeBlock>
<p><em>Paragraphs 3 and 4 were deleted.</em> <br /></p>


#### Dynamic Semantics

<MarginText>5/3</MarginText>
<MarginInfo items={["AI05-0264-1"]} />
<p>For the execution of an <code><a href="../AA-5/AA-5.3#S0175">if_statement</a></code>, the <code><a href="../AA-4/AA-4.5#S0150">condition</a></code> specified after <strong>if</strong>, and any <code><a href="../AA-4/AA-4.5#S0150">condition</a></code>s specified after <strong>elsif</strong>, are evaluated in succession (treating a final <strong>else</strong> as <strong>elsif</strong> True <strong>then</strong>), until one evaluates to True or all <code><a href="../AA-4/AA-4.5#S0150">condition</a></code>s are evaluated and yield False. If a <code><a href="../AA-4/AA-4.5#S0150">condition</a></code> evaluates to True, then the corresponding <code><a href="../AA-5/AA-5.1#S0166">sequence_of_statements</a></code> is executed; otherwise, none of them is executed. <br /></p>

<AnnotatedOnly>
<MarginText>5.a</MarginText>
<Admonition type="aarm" aarm="ramification">
<strong></strong>The part about all evaluating to False can't happen if there is an <strong>else</strong>, since that is herein considered equivalent to <strong>elsif</strong> True <strong>then</strong>. <br /></Admonition>
</AnnotatedOnly>


#### Examples

<MarginText>6</MarginText>
<p><em>Examples of if statements:</em> <br /></p>

<MarginText>7</MarginText>
<CodeBlock language="ada">
if Month = December and Day = 31 then{"\n"}   Month := January;{"\n"}   Day   := 1;{"\n"}   Year  := Year + 1;{"\n"}end if;{"\n"}
<MarginText>8</MarginText>
if Line_Too_Short then{"\n"}   raise Layout_Error;{"\n"}elsif Line_Full then{"\n"}   New_Line;{"\n"}   Put(Item);{"\n"}else{"\n"}   Put(Item);{"\n"}end if;{"\n"}
<MarginText>9</MarginText>
if My_Car.Owner.Vehicle /= My_Car then            -- see <a href="../AA-3/AA-3.10#Subclause_3.10.1">3.10.1</a>{"\n"}   Report ("Incorrect data");{"\n"}end if;{"\n"}

</CodeBlock>
