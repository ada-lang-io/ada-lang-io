#### Dynamic Semantics

<MarginText>1</MarginText>
<p>It is not specified by the language whether the same external file can be associated with more than one file object. If such sharing is supported by the implementation, the following effects are defined: <br /></p>

<MarginText>2</MarginText>
<ul>
<li>Operations on one text file object do not affect the column, line, and page numbers of any other file object.<br /></li>
<MarginText>3/1</MarginText>
<MarginInfo items={["AI95-00050-01"]} />
<li><em>This paragraph was deleted.</em>{"{"}<em>8652/0057</em>{"}"} {"{"}<em>{"AI95-00050-01"}</em>{"}"} <br /></li>
<MarginText>4</MarginText>
<li><br />For direct and stream files, the current index is a property of each file object; an operation on one file object does not affect the current index of any other file object.<br /></li>
<MarginText>5</MarginText>
<li>For direct and stream files, the current size of the file is a property of the external file. <br /></li>
</ul>
<MarginText>6</MarginText>
<p>All other effects are identical. <br /></p>
