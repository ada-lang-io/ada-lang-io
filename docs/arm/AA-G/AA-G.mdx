<MarginText>1</MarginText>
<p>The Numerics Annex specifies <br /></p>

<MarginText>2</MarginText>
<ul>
<li>features for complex arithmetic, including complex I/O;<br /></li>
<MarginText>3</MarginText>
<li>a mode (&ldquo;strict mode&rdquo;), in which the predefined arithmetic operations of floating point and fixed point types and the functions and operations of various predefined packages have to provide guaranteed accuracy or conform to other numeric performance requirements, which the Numerics Annex also specifies;<br /></li>
<MarginText>4/5</MarginText>
<MarginInfo items={["AI12-0445-1"]} />
<li>a mode (&ldquo;relaxed mode&rdquo;), in which there are no accuracy or other numeric performance requirements to be satisfied, as for implementations not conforming to the Numerics Annex;<br /></li>
<MarginText>5/2</MarginText>
<MarginInfo items={["AI95-00296-01"]} />
<li>models of floating point and fixed point arithmetic on which the accuracy requirements of strict mode are based;<br /></li>
<MarginText>6/2</MarginText>
<MarginInfo items={["AI95-00296-01"]} />
<li>the definitions of the model-oriented attributes of floating point types that apply in the strict mode; and<br /></li>
<MarginText>6.1/2</MarginText>
<MarginInfo items={["AI95-00296-01"]} />
<li>features for the manipulation of real and complex vectors and matrices. <br /></li>
</ul>

#### Implementation Advice

<MarginText>7/3</MarginText>
<MarginInfo items={["AI05-0229-1"]} />
<p>If Fortran (respectively, C) is widely supported in the target environment, implementations supporting the Numerics Annex should provide the child package Interfaces.Fortran (respectively, Interfaces.C) specified in <a href="/docs/arm/AA-B/">Annex B</a> and should support a <i>convention{"_"}</i><code><a href="/docs/arm/AA-2/AA-2.3#S0002">identifier</a></code> of Fortran (respectively, C) for the Convention aspect (see <a href="/docs/arm/AA-B/">Annex B</a>), thus allowing Ada programs to interface with programs written in that language. <br /></p>

<AnnotatedOnly>
<MarginText>7.a.1/2</MarginText>
<Admonition type="aarm" aarm="implementation-advice">
<b></b>If Fortran (respectively, C) is supported in the target environment, then interfacing to Fortran (respectively, C) should be supported as specified in <a href="/docs/arm/AA-B/">Annex B</a>.<br />
</Admonition>
</AnnotatedOnly>
